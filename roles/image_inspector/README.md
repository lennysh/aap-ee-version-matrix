Image Inspector
=========

Inspects container images for Ansible versions, Python versions, system packages, Ansible Collections, and Python packages. Discovery and inspection results are saved as one YAML file per digest under `images/<image_path>/vars/<digest-hex>.yml`.

Role Variables
--------------

```yaml
# Defaults shown below

# A list of container image paths to query
image_inspector_image_paths:
  - "registry.redhat.io/ansible-automation-platform/ee-minimal-rhel8"
  - "registry.redhat.io/ansible-automation-platform/ee-minimal-rhel9"
  # ...

# A list of string patterns to exclude from tag discovery results
image_inspector_exclude_patterns:
  - "-source"
  - "sha256"

# The directory where the output YAML files will be saved
image_inspector_output_dir: "{{ playbook_dir }}/images"

# Subdirectory under each image path where per-digest vars files are stored
image_inspector_vars_subdir: vars

# Controls whether to remove digest vars files when they are no longer
# found in the remote registry. Set to true for 'prune' functionality.
image_inspector_prune_images: "{{ prune_images | default(false) }}"
```

Output layout
-------------

```
images/<registry>/<namespace>/<image>/
  vars/
    <digest-hex>.yml    # one file per image digest
  README.md             # generated by md_generator
```
